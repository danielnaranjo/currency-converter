## Set up for development
Make sure you have [npm](https://www.npmjs.com/) [Node.JS](https://nodejs.org/), then:
```
npm install
npm start
```
This should start a new server on `http://localhost:3000/`.

## Build
```
npm run build
```
Built files are located in `./dist`.
